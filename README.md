# Django Web Application - Netflix Dashboard Showcase

## 📌 Overview
This project is a Django-based web application developed to showcase a Netflix Power BI Dashboard. The website provides a simple and interactive interface for presenting the dashboard and project information.

## ✨ Features
- 🏠 Home Page
- ℹ️ About Us
- 📊 Dashboard Page
- 📞 Contact Us Page

## 🛠️ Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite

## 🚀 How to Run

```bash
pip install -r requirements.txt
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
